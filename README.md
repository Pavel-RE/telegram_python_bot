# telegram_python_bot

Телеграмм-боты на python:

Decision-maker_bot.py
Бот, который помогает принять решение.

json_bot.py
Бот, который приводит JSON-файл в надлежащий вид или выдает сообщение об ошибке.

Перед установкой
!pip install pyTelegramBotAPI
